# Alpha Terminal 사용자 시나리오 — 온보딩 & 메뉴 가이드

> 작성일: 2026-04-18  
> 대상: 팀원(개발/기획), 신규 사용자, 기존 사용자  
> 목적: 로그인 후 앱을 어떻게 써야 하는지 첫 흐름을 정의하고, 각 메뉴의 역할을 명확히 공유한다.

---

## 왜 이 문서가 필요한가

기존에는 로그인 후 무엇을 해야 하는지 명확한 안내가 없었다.  
Alpha Terminal은 **관심종목 설정**이 되어야만 브리핑·AI 분석·투자판단 등 핵심 기능이 의미 있게 동작한다.  
사용자가 이를 모른 채 다른 메뉴를 먼저 탐색하면 "왜 내용이 없지?" 하는 혼란을 겪는다.

---

## 메뉴 구조 개요

| 메뉴 | 경로 | 한 줄 설명 |
|---|---|---|
| 홈 | `/` | 관심종목 기반 AI 브리핑 + 주요 지표 요약 |
| 대시보드 | `/dashboard` | 관심종목 종합 분석 실행 및 결과 확인 |
| 관심종목 | `/watchlist` | 관심 종목 추가/삭제 및 현황 확인 |
| 투자판단 | `/invest` | 특정 종목에 대한 AI 멀티에이전트 분석 |
| 주식추천 | `/stock-recommendation` | AI 기반 종목 추천 |
| 뉴스 | `/news` | 종목 관련 뉴스 및 공시 |
| 영상 | `/youtube` | 종목 관련 YouTube 영상 및 감성 지표 |
| 게시판 | `/board` | 커뮤니티 의견 공유 |
| 프로필 | `/profile` | 관심종목 관리, 상호작용 이력, 개인 설정 |

## 시나리오 1 — 신규 사용자 (최초 로그인)

```
[로그인]
    ↓
[홈 화면]
  → 브리핑이 비어있음 ("관심종목을 먼저 설정해주세요" 상태)
    ↓
[관심종목 설정] ← 핵심 진입점
  → 네비게이션 바 "관심종목" 또는 홈 빈상태 CTA 버튼 클릭
  → 종목 검색 후 추가 (예: 삼성전자, 카카오)
    ↓
[홈으로 이동]
  → 설정한 종목 기반으로 AI 브리핑 자동 생성
  → 오전 7시 브리핑 또는 최신 분석 확인
    ↓
[대시보드]
  → "분석 시작" 버튼 → 관심종목 일괄 파이프라인 실행
  → AI 분석 요약, 감성 지표, 종목별 요약 카드 확인
    ↓
[투자판단 / 뉴스 / 영상 등 자유 탐색]
```

**핵심 포인트**  
관심종목이 없으면 홈·대시보드·투자판단 모두 빈 화면이다.  
신규 사용자에게는 첫 화면에서 관심종목 설정을 유도하는 CTA가 반드시 필요하다.

## 시나리오 2 — 기존 사용자 (재방문)

```
[로그인]
    ↓
[홈 화면]
  → 오전 7시에 자동 생성된 AI 브리핑 확인
  → 관심종목 변동 요약, 시장 전반 감성 지표 확인
    ↓
[필요 시 세부 탐색]
  ├── 특정 종목 궁금 → 투자판단 (/invest) → 질의 입력 → 실시간 AI 분석
  ├── 공시/뉴스 확인 → 뉴스 (/news)
  ├── YouTube 여론 확인 → 영상 (/youtube)
  └── 관심종목 변경 → 관심종목 (/watchlist) 또는 프로필 (/profile)
    ↓
[대시보드] (필요 시)
  → 오늘 분석을 직접 실행하거나 최근 결과 재조회
```

**핵심 포인트**  
재방문 사용자의 주요 진입점은 **홈 브리핑**이다.  
브리핑을 보고 → 관심 있는 종목을 클릭하거나 → 투자판단으로 심층 분석하는 흐름.

```
  → 동일하게 종목 추가 후 홈으로 복귀
```

## 메뉴별 상세 시나리오

### 홈 (`/`)
- **주목적**: 오늘의 시장 요약 + 관심종목 AI 브리핑
- **비어있을 때**: 관심종목 설정 CTA 표시 필요
- **브리핑 생성 시점**: 매일 오전 7시 (KST) 자동 실행

### 대시보드 (`/dashboard`)
- **주목적**: 관심종목 전체에 대한 분석 파이프라인 실행
- **사용 방법**: "분석 시작" 버튼 클릭 → 뉴스/공시/YouTube 데이터 수집 → AI 요약 생성
- **결과 보기**: 종목별 AI 분석 요약 카드, 감성 점수, 태그

### 투자판단 (`/invest`)
- **주목적**: 특정 종목에 대해 자유 질의 → AI 멀티에이전트 실시간 분석
- **사용 방법**: "삼성전자 지금 사도 될까?" 입력 → 에이전트 분석 로그 실시간 확인 → 최종 판단
- **캐시**: 동일 종목은 1시간 내 재질의 시 즉시 반환

### 관심종목 (`/watchlist`)
- **주목적**: 종목 추가/삭제 + 현재가·등락 확인
- **중요성**: 앱의 모든 개인화 기능이 이 목록을 기반으로 동작

### 프로필 (`/profile`)
- **주목적**: 관심종목 관리 + 상호작용 이력(클릭한 카드 등) 확인
- **추가 예정**: 알림 설정, 분석 히스토리
